Arithmetic Means Calculator

Find Term

Formula: aₙ = a₁ + (n - 1) * d

Where:

aₙ = Term number n, a₁ = First term, d = Common difference, n = Term number

Substitute the values into the formula:

aₙ = (First term) + (Term number - 1) * (Common difference)

Find d

Formula: d = (Aₙ - a₁) / (n - 1)

Where:

Aₙ = Last term, a₁ = First term, n = Number of terms

Substitute the values into the formula:

d = (Last term - First term) / (Number of terms - 1)

Sum of All Terms

Formula: S = (n / 2) * (2 * a₁ + (n - 1) * d)

Where:

S = Sum of all terms, a₁ = First term, d = Common difference, n = Number of terms

Substitute the values into the formula:

S = (Number of terms / 2) * (2 * First term + (Number of terms - 1) * Common difference)

Geometric Means Calculator

Find Term

Formula: aₙ = a₁ * r^(n - 1)

Where:

aₙ = Term number n, a₁ = First term, r = Common ratio, n = Term number

Substitute the values into the formula:

aₙ = First term * (Common ratio)^(Term number - 1)

Find r

Formula: r = (Aₙ / a₁)^(1 / (n - 1))

Where:

Aₙ = Last term, a₁ = First term, n = Number of terms

Substitute the values into the formula:

r = (Last term / First term)^(1 / (Number of terms - 1))

Sum of All Terms

Formula: S = a₁ * (r^n - 1) / (r - 1)

Where:

S = Sum of all terms, a₁ = First term, r = Common ratio, n = Number of terms

Substitute the values into the formula:

S = First term * ((Common ratio^Number of terms - 1) / (Common ratio - 1))
